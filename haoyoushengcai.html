# 蚝油生菜

耗油生菜中熬蒜蓉的方法还可以用在 蒜蓉油麦菜，蒜蓉秋葵 中

# 熬蒜蓉

  • 大蒜切碎，取 1/2 用水冲
  • 锅中添油，下入用水冲过的蒜末，炒至上色
  • 加蚝油，酱油
  • 加味精，糖调味
  • 加入另一半蒜，搅匀盛出

TIP

用水冲后的蒜碎熬出来不容易苦

# 煮生菜

  • 起锅烧水，加入盐和油
  • 水开后下生菜烫 3 秒
  • 盛出生菜，淋上上一步熬好的蒜蓉即可

TIP

水中加入盐和油可以使生菜保持翠绿
